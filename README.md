# Baikonur District Shooter — ISO Edition (Telegram Mini App, MVP без сервера)

Это псевдо‑3D **изометрический шутер** для Telegram Mini Apps (HTML5 Canvas).

## Фичи
- Изометрическая картинка: плитка, тени, «объёмные» дома.
- Управление: ПК (WASD + мышь + клик), телефон (джойстик + кнопки).
- Типы врагов: обычный, танк, снайпер, камикадзе.
- Лут: аптечки/щит/монеты.
- «Зоны района»: мини‑квесты в пределах зоны (чистка, удержание).
- Магазин: оружие/снаряга/скины/бусты, всё с иконками.
- Сохранение прогресса: localStorage + опционально Telegram CloudStorage.

## Деплой
GitHub Pages: Settings → Pages → Branch `main` → Folder `/docs`.
BotFather: `/setmenubutton` → Web App → URL Pages.

MVP без сервера: одиночка и локальный прогресс.
